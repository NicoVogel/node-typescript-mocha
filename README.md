# This is a template for NodeJS TypeScript Mocha based TDD projects

This template provides:

- a devcontainer for an consistent environment
- linting via eslint
- formatting via prettier

## Repo Structure

The intended structure:

````
src/ 
  - foo.ts
  - <code>

test/
  - foo.test.ts
  - <tests>

dist/
  - foo.js
  - <compiled js>
````
